# Oauth 2.0 for 1C-Bitrix with ActionFilter

This module enables website authorization using Oauth 2.0. It includes administrative pages for managing clients. Each
client has its own key pair, which can be used to obtain a token via the API (1C-Bitrix module controllers). This token
can then be used in any of your controllers as an ActionFilter when defining Action configurations in your module
controller.

## Installation

You can install it by default 1C-Bitrix methods from
marketplace [here](https://marketplace.1c-bitrix.ru/solutions/agrebnev.oauth20/)
